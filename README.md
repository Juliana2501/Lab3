## Управление историей цен товара

### Описание проекта
Программа на Java для управления товарами с историей изменения цен.

### Функционал
1. Создание товара с себестоимостью и ценой
2. Автоматическая коррекция цены ниже себестоимости
3. Изменение цены товара
4. Добавление наценки в процентах
5. Применение скидки в процентах
6. Изменение цены на фиксированное значение
7. Расчет средней цены
8. Расчет средней наценки
9. Расчет возможной выручки
10. Хранение истории цен
11. Вывод истории цен

## Классы программы

### Класс `Product` - основной класс товара

**Назначение:**  
Представляет товар в системе. Хранит информацию о товаре, управляет ценами и историей их изменений.

**Зачем нужен:**  
- Хранит все данные о товаре в одном месте
- Содержит логику проверки цен на соответствие себестоимости
- Управляет историей изменений цен
- Выполняет все расчеты, связанные с ценами

**Поля класса:**
- `name` - название товара
- `costPrice` - себестоимость (минимальная цена)
- `currentPrice` - текущая цена продажи
- `priceHistory` - список для хранения истории цен
- `MAX_HISTORY_SIZE` - максимальное количество записей в истории (10)

**Основные методы:**
- `Product(name, costPrice, initialPrice)` - создает товар с начальной ценой
- `Product(name, costPrice)` - создает товар без начальной цены (цена = себестоимости)
- `getInfo()` - выводит информацию о товаре
- `setPrice(newPrice)` - устанавливает новую цену
- `addMargin(percent)` - добавляет наценку в процентах
- `applyDiscount(percent)` - применяет скидку в процентах
- `changePriceByFixed(amount)` - изменяет цену на фиксированную сумму
- `getAveragePrice()` - вычисляет среднюю цену
- `getAverageMargin()` - вычисляет среднюю наценку
- `calculateRevenue(quantity)` - рассчитывает выручку
- `printPriceHistory()` - выводит историю цен
